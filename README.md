## STEPS TO GET STARTED

### 01. Install Python

Windows

- Go to [PYTHON.ORG](https://python.org/downloads/) and click on Download Python 3.XX.XX
- Make sure you Check Mark **Add Python 3.XX to PATH** while installing.

Linux

- Install using command - `sudo apt install python3 python3-pip -y`

### 02. Clone the Repo

- Install Git from https://git-scm.org if not done already.
- Open Terminal and run command `git clone https://github.com/stayshikshit/flask-blog.git -b flask-introduction`
- Open the Project with VS Code `code ./flask-blog`

### 03. Install Flask

- Use command `pip install flask`

### 04. Run the Application

- Use command `python main.py` from the project folder in Terminal.
